В математике, в частности в линейной алгебре, кососимметричная матрица (также известная как антисимметричная), является квадратная матрица A, транспонированная и отрицательная.     Это удовлетворяет уравнению A = −AT. Если элемент i-ой строки и j-ого столбца aij, т.е. A = (aij) тогда симметричное значение становится равным aij = −aji.

Вы должны определить, является ли указанная квадратная матрица кососимметричной или нет.

Вы можете найти более подробную информацию о кососимметричных матрицах на странице Википедии.

Входные данные: Квадратная матрица в виде списка списков целых чисел.

Выходные данные: Является ли матрица кососимметричной как логическое.

Пример:

    [-2, -1,  0]]) == True
symmetric([
    [ 0,  1, 2],

    [-2, -1, 0]]) == False
symmetric([
    [ 0,  1, 2],
    [-1,  0, 1],
    [-3, -1, 0]]) == False
{% endif %}

Как это используется: Кососимметричные матрицы могут быть полезны для векторного произведения, операций по математике, используемых для расчета движения сил. Матрицы основа линейной алгебры и векторной графики.

Предусловия: 0 < N < 5
